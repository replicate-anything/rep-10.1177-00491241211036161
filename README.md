## Build display artifacts

From this repository root:

```r
library(replicateEverything)
options(
  replicateEverything.registry_root = "../registry",
  replicateEverything.use_sibling_packages = TRUE
)
replicateEverything::build_study_artifacts(".", install_deps = TRUE)
```

## Prepare for registry (writes `registry/` stub files)

```r
replicateEverything::prepare_folder_paper(
  ".",
  build_artifacts = FALSE,
  registry_root = "../registry"
)
```

This creates:

- `registry/replication.yml` → copy to `registry/studies/10.1177_00491241211036161.yml`
- `registry/index.csv` → merge into the registry repo `index.csv`

## Sync to registry checkout

```r
options(replicateEverything.registry_root = "../registry")
replicateEverything::sync_folder_paper(".")
```

Or use `add_folder_paper(".")` to prepare + sync in one step when the registry is a local sibling folder.
